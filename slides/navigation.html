Navigation

  • Go back a slide by pressing the left arrow key
  • Go forward a slide by pressing the right arrow key or spacebar
  • Go to the first slide by holding the control/command key and pressing the left arrow key
  • Go to the last slide by holding the control/command key and pressing the right arrow key
  • Pressing a number key will navigate to the specified slide (e.g. pressing 2 will go to the second slide), except for the 9 key, which goes to the last slide
  • The list of slides and their load times can be viewed by pressing s, and can be closed by pressing the key again
  • The list of general load times functions the same way as the list of slides, but uses the d key
  • If you're lost, pressing h always brings you back to the slides
