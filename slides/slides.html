Slides

Slides are individual HTML files stored within the slide directory, which can be customised in the config.

This slide's code looks something akin to this:

<div data-next="docs">  <h1>Slides</h1>  <p>Slides are individual HTML files stored within the slide directory, which can be customised in the config.</p>  <p>This slide's code looks something akin to this:</p>  <code>Recursion...</code>  <p>> Docs</p> </div>

Slides are an individual div element. If there is a following slide, link to it by setting the data-next of the div to the name of the next slide (the file name, excluding the .html).
