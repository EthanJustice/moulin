Tour - Slides

Here's some useful knowledge about slides (if you're looking for examples of slides, this slide's code can be found here).

  • Slides have scoped styles (implementation notes) - that is, any style element within a slide will be scoped to that slide only (for example, this is a b element that has styles unique to this slide - check the next slide to see that it works)
  • Any element works - even iframes

A guide to what happens before a slide is added:

  1. Slide content is fetched based on the data-next attribute of each slide (or loaded from the cache)
  2. The content is run through the scoped CSS parser
  3. If there's a permalink, the proper slide is navigated to
