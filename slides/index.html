Welcome, friends.

Moulin is somewhat minimal on purpose, so navigation is based on keyboard shortcuts.

To go to the next slide, use the right arrow key or spacebar.

To go back, use the left arrow key.

Alternatively, you can just press h and return to the slides from anywhere.

To go all the way to the last slide, hold the control/command key and press the right arrow key. If you want to do the opposite, hold control/command and press the left arrow key.
