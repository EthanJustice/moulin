Moulin is designed to be fast, efficent, and readily extensible.

As such, it is somewhat minimal on purpose, so navigation is based on keyboard shortcuts.

To go to the next slide, use the right arrow key or spacebar.

To go back, use the left arrow key.

Alternatively, you can use the number keys to navigate to the slide matching that key, or the last one, if you use the 9 key.

To go all the way to the last slide, hold the control/command key and press the right arrow key. If you want to do the opposite, hold control/command and press the left arrow key.
